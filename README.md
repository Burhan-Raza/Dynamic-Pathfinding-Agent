Dynamic Pathfinding Agent
Description

This project implements A* and Greedy Best-First Search algorithms for dynamic pathfinding using Pygame.

Requirements

Python 3.x

Pygame

Install Dependencies

Install pygame using:

pip install pygame

If pip does not work, try:

python -m pip install pygame

How to Run the Project

Open terminal inside the project folder.

Run the following command:

python main.py

Make sure your file name matches your actual Python file.

Features

A* Search

Greedy Best-First Search

Manhattan and Euclidean heuristics

Dynamic obstacle generation

GUI visualization
